# My Academic Portfolio

A professional e-portfolio showcasing my Level 6 Computer Science journey at JKUAT.

## Technologies Used
* **HTML5:** Semantic structure and content.
* **CSS3:** Custom styling and responsive layout (Mobile-friendly).
* **JavaScript:** Interactive elements and UI logic.

## How to Run Locally
1. Clone the repository: `git clone https://github.com/HeatSpeed/My-Academic-Portfolio.git`
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser.
